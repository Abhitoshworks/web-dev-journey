# Web Development Practice

A dedicated repository for tracking my web development learning journey. This collection contains practice code, mini-projects, and exercises completed while following tutorials.

The goal of this repository is to build a solid foundation in web development. By organizing my practice files here, I can easily look back at my code to review concepts and track my progress over time.

To keep things organized, the code is divided into topic-specific folders:

* `/css-styling` - Layouts, Flexbox, Grid, and animations.
* `/js-fundamentals` - Variables, functions, DOM manipulation.
* `/mini-projects` - Small, combined projects applying multiple skills.
